This project analyzes the performance of probabilistic modeling techniques using a synthetic dataset generated from a hierarchical stochastic simulation framework. The dataset contains both numerical variables—such as reconstruction accuracy, masking rate, and computational runtime—and categorical variables, including model type (Bayesian inference, Hidden Markov Models, and Graph Neural Networks) and dataset configuration.

The primary objective is to investigate how model choice and data degradation impact reconstruction accuracy. Using linear modeling techniques, the analysis explores relationships between masking rate and accuracy while accounting for differences across model types. Interaction effects between model type and masking rate are also examined to determine whether certain models are more robust under increasing levels of missing data.

Additionally, analysis of variance (ANOVA) is used to compare mean performance across categorical groupings, providing insight into statistically significant differences between modeling approaches. This study demonstrates how classical statistical methods can be applied to evaluate modern machine learning systems within a controlled, reproducible experimental framework.
